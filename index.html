TrimestrePro
 

TrimestrePro est un moyen simple de faire le point - en dix minutes ou sur plusieurs sessions. Revenez sur le dernier trimestre et analysez ce que vous avez accompli. Ensuite, concentrez vous sur le prochain trimestre et déterminez vos objectifs.

Commencez avec un guide, un modèle imprimable et des rappels trimestriels

Pourquoi

Un trimestre, c'est environ 91 jours soit 2184 heures. C’est beaucoup de temps pour faire des progrès substantiels dans n'importe quel domaine. Utilisez TrimestrePro pour analyser les trois derniers mois. Vous comprendrez votre progression et améliorerez vos méthodes d'apprentissage. Quatre check-ins par an vous permettent d'analyser, comparer et ajuster assez souvent pour rester sur la bonne voie et booster votre productivité.

Si vous avez des projets plus ambitieux que le trimestre, un check-in par trimestre vous donne un délai dans lequel vous pourrez élaborer des plans plus concrets.

Si vous effectuez déjà un point hebdomadaires, TrimestrePro vous aidera à vous concentrer sur les stratégies globales qui guideront vos semaines.

Comment

1. Trouver un espace neutre

Pour de bons résultats, il vaut mieux s'échapper si vous le pouvez - même si c’est dans le café du coin.

2. Rejoignez la newsletter TrimestrePro

S'inscrire ici pour être notifié quand le trimestre approche.

3. Faites le check-in

a. Faites le point

Une réflexion sur les 3 derniers mois. Ce que vous avez fait - les événements, les expériences, les promotions, les valeurs qui ont été significatives pour vous. Comment ça s'est passé - ce qui s'est bien passé, ce qui aurait pu mieux se passer et ce que vous avez appris.

b. Donnez vous des objectifs

Envisager les 3 prochains mois. Ce que vous visez - objectifs, habitudes, expériences. Comment vous allez les aborder et ce que vous comptez en tirer.

c. Résumé

Distillez le tout en deux-trois phrases pour voir votre trimestre en un seul coup d'œil. C'est un excellent point de référence pour le prochain check-in.
